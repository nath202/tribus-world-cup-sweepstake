Paste the draw app's results — CSV, the copied clipboard text, or one player per line. Then preview and apply.

Accepted formats
• Name: t1, t2, t3, t4 per line
• CSV wide — header row then Participant,Team1..Team4
• CSV long — Participant,Team rows, grouped automatically
Team names are matched loosely (e.g. "South Korea" → Korea Republic, "Turkey" → Türkiye).

Live data source

Set once, then it runs itself

Once this points at the World Cup and auto-refresh is on, the dashboard updates on its own — no clicking. Best results when the file is hosted (e.g. GitHub Pages); from a local file the browser may block the connection.

How "self-managed" actually works
1. Click Find the World Cup — it lists matching leagues from TheSportsDB; pick the 2026 one and the ID fills itself in.
2. Click Test connection — it reports how many fixtures/results it can see, so you know in seconds whether live data exists for that league/season.
3. Tick Auto-refresh, Save — from then on it polls and updates with no input from you.

If the test finds nothing, the free tier doesn't have WC2026 under that league yet — the Teams tab remains a full manual fallback, and your edits are never overwritten. For reliable live data, host the file rather than opening it locally.
